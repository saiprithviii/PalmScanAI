# PalmScan AI

PalmScan AI is a deep learning–based web application developed for detecting nutrient deficiencies in oil palm leaves using image classification.

The system allows users to upload a palm leaf image and receive:

- Predicted nutrient deficiency
- Confidence score
- Recommended treatment protocol
- Multilingual support interface

## Features

- AI-powered nutrient deficiency detection
- Real-time image prediction
- Modern Streamlit dashboard interface
- Confidence visualization
- Treatment recommendation system
- Multi-language support:
  - English
  - Telugu
  - Hindi
  - Tamil
  - Kannada

## Detectable Conditions

- Boron Deficiency
- Nitrogen Deficiency
- Kalium Deficiency
- Magnesium Deficiency
- Healthy Leaf

## Tech Stack

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow
- CSS
